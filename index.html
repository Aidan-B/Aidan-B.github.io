TeamUp

TeamUp is a Discord bot that was created for the discord hackweek competition. TeamUp is designed to help people schedule groups to "Team up" with each other so that they can play together at the same time. The bot creates groups based on the user desired game and time, and will notify players with a private message when it is time to play.
